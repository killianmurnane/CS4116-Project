Only logged-in users can access the messaging page. The left panel lists your matches and selecting a match loads messages for that conversation ordered by oldest first. Match display names are built from the other user's profile name when available.

Messages can only be sent to valid match ids that belong to you. Submitting a message requires both a selected match id and non-empty message text. Invalid or unauthorized match ids redirect with an error. Empty or invalid input also redirects with an error. Successful messages redirect back to the same match conversation. Messages containing phone numbers are automatically replaced with hashes to ensure security and safety in line with site policy.

Messages sent by you are labelled You and styled differently. Messages from the other participant are labelled Them. If you have no matches the page tells you to like profiles first. If a match has no messages yet the page prompts you to start the conversation.
