The admin page is restricted to admin users only. Non-admin users are redirected away.

Admins can search users by email or full name. The user list includes role email and location when available. The selected user record is shown in a detail panel.

Admin actions include changing another user's role to admin standard or banned editing another user's profile fields clearing a user's profile description and deleting messages sent by the selected user. Admins cannot change their own account role from the admin form.

Profile updates validate location ids and accept optional fields such as family name gender date of birth description and preferred sessions. The admin view shows recent messages sent by the selected user limited to 12 messages. Success and failure states are shown as alerts on the page.
